# AI i frontendutveckling: hot eller transformation?

Detta arkiv innehåller källkoden för de praktiska experimenten i examensarbetet "AI i frontendutveckling: hot eller transformation?". 

### Syfte
Syftet med detta projekt är att undersöka hur moderna AI-modeller hanterar digital tillgänglighet (WCAG 2.1) och kodkvalitet (Clean Code) vid generering av frontend-komponenter. Koden i detta arkiv är den direkta outputen från de tester som beskrivs i rapporten.

### Innehåll
Arkivet är uppdelat efter de två modeller som har utvärderats:
- **GPT-5.5:** Innehåller den monolitiska källkoden (App.tsx/App.css).
- **Opus-4.7 ** Innehåller den modulära källkoden uppdelad i separata komponenter.

### Användning
Koden är avsedd för granskning och utvärdering. För att köra projektet lokalt krävs en React-miljö med Vite.


# AI in Frontend Development: Threat or Transformation?

This repository contains the source code for the empirical experiments conducted as part of the bachelor thesis "AI in Frontend Development: Threat or Transformation?".

### Purpose
The objective of this project is to evaluate how modern AI models handle web accessibility (WCAG 2.1) and code quality (Clean Code) when generating frontend components. The code provided here is the raw output from the tests detailed in the thesis report.

### Contents
The repository is organized based on the two models evaluated:
- **GPT-5.5:** Contains the monolithic source code (App.tsx/App.css).
- **Opus-4.7:** Contains the modular source code split into separate components.

### Usage
This code is intended for review and audit purposes. To run the project locally, a React environment with Vite is required.
